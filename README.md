# 💖 Barbie Fan Website

This project is a simple **Barbie themed fan website** created using **HTML and CSS**.

## 🎀 About The Project

I designed this website as a fun fan page inspired by the Barbie universe.
The goal of this project is to practice **basic web design, layout, and CSS styling**.

## 🛠️ Technologies Used

* HTML5
* CSS3

## ✨ Features

* Barbie themed colors and design
* Image gallery section
* Simple and clean layout
* Beginner friendly structure

## 📌 Purpose

This project was created to improve my **front-end development skills** and learn how to design aesthetic websites using CSS.
